## About

Python script for generating push notifications for iOS simulators.

## Prerequisites

Python 3

## Setup

Run make setup on your terminal.

## Usage
1. Open project terminal
2. Run `make run`
3. Prompt required information:
    - Device Token: Device Unique Identifier
    - Bundle Identifier: App Unique Identifier
    - Title: Push Notification Title
    - Body: Push Notification Body
    - Action: Push Notification Action
